Cara Melakukan Transaksi dengan QRIS

Ikuti langkah-langkah berikut untuk melakukan pembayaran menggunakan QRIS:

Melalui DANA:

  1. Buka aplikasi DANA di ponsel Anda.
  2. Pilih menu Scan QR.
  3. Arahkan kamera ponsel Anda ke QRIS code di atas.
  4. Masukkan jumlah pembayaran sesuai dengan layanan yang Anda pilih.
  5. Konfirmasi pembayaran dan pastikan transaksi berhasil.

Melalui GoPay:

  1. Buka aplikasi Gojek di ponsel Anda.
  2. Pilih menu Bayar.
  3. Arahkan kamera ponsel Anda ke QRIS code di atas.
  4. Masukkan jumlah pembayaran sesuai dengan layanan yang Anda pilih.
  5. Konfirmasi pembayaran dan pastikan transaksi berhasil.

Jika Anda menggunakan platform lain, pastikan untuk mengikuti langkah serupa sesuai dengan aplikasi pembayaran Anda.
